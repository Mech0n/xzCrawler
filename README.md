# xzCrawler

> Save a file for xianzhi. 

### How to Use?

⚠️: Change the range in `main.py` first.

- pip install the requirments.
    ```shell
    pip install -r requirements.txt
    ```
- Run the script.
    ```shell
    python main.py
    ```

### Features

- Incremental update
    - Python sqlite : `database.db` stores the requested past-url
